### Быстрый старт
- Клонируйте репозиторий: `git clone https://github.com/AsotikovaSvetlana/mail-box.git`
- Перейдите в папку с проектом: `cd mail-box`
- Установите зависимости: `npm install`
- Запустите проект: `npm start`
